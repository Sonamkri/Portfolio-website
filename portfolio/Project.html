Diabetes Predictor using Machine Learning

This project aims to predict whether a patient is likely to have diabetes based on various health-related metrics. It uses machine learning techniques trained on the Pima Indians Diabetes Dataset to provide predictive analysis. The system includes data preprocessing, visualization, model training, and evaluation using algorithms like Logistic Regression and Random Forest. Python libraries such as Pandas, NumPy, Scikit-learn, Matplotlib, and Seaborn were used. A user-friendly web interface was built using Streamlit to allow real-time predictions. The model's performance is evaluated using accuracy scores, confusion matrix, and ROC curves.
